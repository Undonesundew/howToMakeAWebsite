Getting Started

We will be using Github Pages along with Visual Studio Code.

Setting up Github Pages.

Head to Github.com, and if you don't have one, create an acount. And if you don't have the github desktop app, download it here. You can use any word editor, like notepad, but downloading Visual Studio Code will be worth your time.

First we will create a repository. In Guthub Desktop Hit Ctrl + N, name your repository, and hit create. This is the folder we will be creating our website in. Now hit Ctrl + P, or press the publish repository button, to add it to the github website, where it will store a copy of the code.

Now, still in the GutHUb desktop app, press Ctrl + Shift + G. This will take you to the repository's page on the GitHub website. Go to the settings tab under the title of the repository, and scroll down until you hit the danger zone. Change the visibility to public. This will allow us to set up github pages.

Still in the settings tab, press the pages button on the side menu, and enable pages. Also set the branch it's deployed from to main. It should look like this.

Setting up Viusal Studio Code

Once you launch VS Code, if you are prompted to sign in, you can use either your GitHub account or your Microsoft account. Or Both. Now go back to GitHub Dekstop, press Ctrl + , (comma) go to integrations, and select Visual Studio Code as your external editor.

Whenever you want to edit your website, go to the GitHub app and hit Ctrl + Shift + A to open your project in VS Code. I recomend using this VS Code extension, which allows you to preview your website live while editing. There will be a button in the top right of VS Code with two rectangles and a magnifing glass. That will show you the live preview.

Finding a template

Go to bootstrapmade.com. Press live demo to see the selected website template as a website. I will be using the Impact template, located near the top. Once you picked one, press download. Once the download is completed, move the contents of the folder in the zip folder, to your website repository on your computer, you can do that by pressing Ctrl + Shift + F in the GitHub app. Now you can start making your website.

Tips to build your own website.

Once you open Index.html, VS Code will look something like this. Probably with different colors. First thing, on line 8, you should see a TITLE tag. That says what the text in the tab at the top of the browser will be. On line 13, is saying what the image beside the title at the top of the browser is. To change it, replace favicon.png in assets > img. Remeber that, as you change things in your file, the numbers of the lines may change; and in most color themes, the gray is the text.
